Open Git Bash.
Change the current working directory to your local project.
Initialize the local directory as a Git repository.
$ git init -b main
Add the files in your new local repository. 
This stages them for the first commit.
$ git add .
# Adds the files in the local repository and stages them for commit. 
To unstage a file, use 'git reset HEAD YOUR-FILE'.

Commit the files that you've staged in your local repository.
$ git commit -m "First commit"
# Commits the tracked changes and prepares
them to be pushed to a remote repository. To remove this commit and modify the file, use 'git reset --soft HEAD~1' and commit and add the file again.
At the top of your repository on GitHub.com's Quick Setup page, click  to copy the remote repository URL.
