Remove the Existing Virtual Environment
$(myenv)cd ~/CRUD
deactivate
rm -rf myenv

Recreate the Virtual Environment
python3.8 -m venv myenv
Activate the New Virtual Environment
source myenv/bin/activate

Navigate to the parent directory of the cloned directory:
cd /path/to/parent/directory
cd CRUD

Remove the cloned directory:
rm -rf cloned_directory_name
rm -rf CRUD


Remove the Non-Empty Directory
rm -rf /home/sunilc/CRUD

# Activate your virtual environment
source /home/sunilc/CRUD/myenv/bin/activate

# Install wheel
pip install wheel

# Upgrade requests to the required version
pip install requests==2.31.0

# Reinstall problematic packages
pip install PyPDF3 svglib --no-cache-dir


Clear Cache Directory:
rm -rf ~/.cache/*

Remove IPython Directory (if not using IPython):
rm -rf ~/.ipython

Check Disk Usage Again:
du -sh ~/*

Clear Package Cache:
pip cache purge

# Remove pip cache
rm -rf ~/.cache/pip


You can use the du command to find large directories or files:
du -ah ~ | sort -rh | head -n 20


Remove Old Log Files
find ~/ -name "*.log" -type f -delete

Remove Unnecessary Dependencies: If you have installed unnecessary packages, 
you can uninstall them:
pip freeze | xargs pip uninstall -y
